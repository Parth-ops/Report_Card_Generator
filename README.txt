Please install the latest version of the following modules and libraries:
1.Matplotlib
2.Numpy
3.pip
4.Pandas
5.Jupyter notebook
6.Pyinstaller

You can run the application in 3 ways:

1. Through cmd: just double click on the 'RunCmd.bat' file. 
Two images will be displayed as u cancel, new students images will get displayed.
You can check for the pdfs in the Report_Cards folder.


2. Through jupyter notebook: double click 'RunNotebook.bat' file.
Select the RepoGen.ipynb file and run all cells.
You can check for the pdfs in the Report_Cards folder.
